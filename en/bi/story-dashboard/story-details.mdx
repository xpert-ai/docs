# Story Details

## Overview

The Story Details page displays complete information about a story, including basic information, content structure, access statistics, etc., making it convenient for you to manage and view stories.

## Story Basic Information

### Basic Information Display

- **Story Title**: The name of the story
- **Story Description**: Detailed description of the story
- **Created At**: The creation time of the story
- **Updated At**: Last modification time
- **Creator**: The creator of the story
- **Project**: The project to which the story belongs

### Story Status

- **Draft**: Being edited, not published
- **Published**: Published and accessible
- **Archived**: Archived and no longer in use

## Story Content

### Chapter List

Display all chapters of the story:
- Chapter title
- Chapter description
- Number of charts included
- Chapter order

### Chart Preview

- Display thumbnails of all charts
- Click to view full-size images
- Support quick editing

## Access Statistics

### Access Data

- **Total Views**: Total number of visits to the story
- **Unique Visitors**: Number of unique visiting users
- **Average Stay Time**: Average time users stay on the page
- **Visit Trend**: Trend chart of visit volume

### User Feedback

- Number of comments
- Number of likes
- Number of shares

## Operations

### Edit Story

Click the "Edit" button to enter edit mode and modify the story content.

### Copy Story

Copy the story to create a duplicate, allowing you to quickly create a new story based on an existing one.

### Delete Story

Delete the story and all its content. This operation cannot be undone.

### Export Story

Export the story as PDF, images, or other formats.

## Permission Management

### Access Permission

- View current permission settings
- Modify access permissions
- Add or remove authorized users

### Collaboration Permission

- Set edit permissions
- Manage collaborators
- Set comment permissions

## Version History

### Version List

View all versions of the story:
- Version number
- Modification time
- Changes made
- Modifier

### Version Comparison

Compare differences between different versions.

### Version Rollback

Roll back to a previous version.
