The menu items are listed in order:

| Name           | Link         | Description                                                     |
| -------------- | ------------ | -------------------------------------------------------------- |
| Open Model     | [Semantic Model](../../models/)       | Go to the semantic model workspace used by the story dashboard |
| New Page       | [New Canvas Page](../workspace/#new-canvas-page)       | Open to create a new page                                      |
| Story Details  | [Story Details](../story-details/) | Open the story details window                                   |
| Advanced Styles | [Advanced Styles](../advanced-styles/) | Open the advanced styles editing window                        |
| Calculated Members | [Manage Calculated Measures and Parameters](../calculations/#manage-calculated-measures-and-parameters) | Open the story's calculated members and parameters editing window |
| Theme Builder  |  | Open the configuration editor window for three themes (Light, Dark, Thin) |
| Templates      | [Template Library](../template/) | Open the story template library                                 |
| Edit           |  | Open the edit submenu, functions include: Save, Copy, Paste, Save As Template |
| Share          | [Share](../share/) | Open the story share window                                    |
| Export         |  | Export the story dashboard as a local file                     |
| Pan            | [Pan](#pan)   | Switch to panning mode                                          |
| Zoom           | [Zoom](#zoom) | Change the canvas size                                          |

## Pan

In pan mode, you can pan the canvas by dragging with the mouse. Users can press the `Alt` key and then drag the canvas with the mouse.

## Zoom

In zoom mode, you can zoom in or out on the canvas by clicking the zoom in or zoom out buttons, or directly using the zoom level in the menu. You can also use shortcuts:

1. **Alt** + **+** Zoom in 10%
2. **Alt** + **-** Zoom out 10%
3. **Alt** + **Esc** Restore to initial state

## Shortcuts

| Shortcut | Description |
| --- | --- |
| `Ctrl + S` | Save current story |
| `Ctrl + Z` | Undo |
| `Ctrl + Shift + Z` | Redo |
| `Alt + -` | Zoom out |
| `Alt + (= or +)` | Zoom in |
| `Alt + 0` | Reset zoom |
| `Alt + Pointer` | Pan |
| `Alt + Esc` | Reset zoom and pan |
