# 自定义 SMTP

## Overview

自定义SMTP功能允许您配置自己的邮件服务器，用于发送系统通知邮件。

## 配置步骤

1. 进入自定义SMTP页面
2. 填写SMTP服务器信息
   - **SMTP服务器**: SMTP服务器地址
   - **端口号**: SMTP端口（通常为25、465或587）
   - **用户名**: SMTP用户名
   - **密码**: SMTP密码
3. 配置安全选项
   - **加密方式**: SSL/TLS
   - **认证方式**: 选择认证方式
4. 测试邮件发送
5. 保存配置

## 配置选项

### SMTP服务器

- **服务器地址**: SMTP服务器地址
- **端口号**: SMTP端口号
- **加密方式**: SSL或TLS

### 认证信息

- **用户名**: SMTP认证用户名
- **密码**: SMTP认证密码

### 发件人信息

- **发件人邮箱**: 系统邮件的发件人邮箱
- **发件人名称**: 发件人显示名称

## 测试配置

配置完成后，可以发送测试邮件验证配置是否正确。
