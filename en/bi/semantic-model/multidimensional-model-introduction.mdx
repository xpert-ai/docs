# 多维模型介绍

## Overview

语义模型是数据分析平台的核心功能之一，用于理解和处理自然语言查询，将用户的问题转换为可执行的数据分析任务。多维模型是语义模型的重要组成部分，用于构建和管理多维数据立方体，支持复杂的OLAP查询和分析。

## 什么是多维模型

多维模型是一种数据组织方式，将数据按照多个维度（如时间、地区、产品等）进行组织，形成一个数据立方体结构。这种结构使得数据分析更加直观和高效。

### 核心概念

- **维度（Dimension）**: 数据的观察角度，如时间、地区、产品类别等
- **度量（Measure）**: 要分析的数值，如销售额、订单量等
- **层次结构（Hierarchy）**: 维度内的层级关系，如年-季度-月-日
- **数据立方体（Cube）**: 多维数据的存储结构

## 主要特性

- **多维度分析**: 支持从多个维度同时分析数据
- **快速查询**: 预计算的数据结构，查询速度快
- **灵活钻取**: 支持上卷（Roll-up）、下钻（Drill-down）、切片（Slice）等操作
- **可视化展示**: 支持多种图表展示方式
- **自然语言理解**: 能够理解用户的自然语言查询

## 快速开始

### 创建多维模型

1. 进入语义模型页面
2. 点击"新建多维模型"按钮
3. 选择数据源
4. 定义维度和度量
5. 设置层次结构
6. 保存并发布模型

### 基本使用示例

```
用户输入: "显示上个月的销售额"
语义模型解析: SELECT SUM(amount) FROM sales WHERE month = LAST_MONTH
```

## 使用场景

- **销售数据分析**: 按时间、地区、产品等维度分析销售数据
- **财务数据分析**: 多维度财务指标分析
- **运营数据分析**: 用户行为、业务指标等多维度分析
- **业务指标监控**: 实时监控关键业务指标

## 模型类型

### SQL多维模型

基于SQL查询的多维模型，适合关系型数据库。

### MDX多维模型

基于MDX（Multidimensional Expressions）的多维模型，提供更强大的多维查询能力。
