# 文档

## Overview

文档微件用于在故事仪表盘中展示文档内容，支持多种文档格式，如PDF、Word、Markdown等。

## Features

- **多格式支持**: 支持PDF、Word、Markdown等格式
- **文档预览**: 支持文档预览功能
- **文档下载**: 支持文档下载功能
- **搜索功能**: 支持文档内容搜索

## Usage

### 添加文档

1. 在故事仪表盘中，点击"添加微件"
2. 选择"文档"
3. 上传或选择文档
4. 配置文档选项
5. 设置显示方式
6. 保存配置

## 支持的格式

### PDF文档

- 支持PDF文件预览
- 支持PDF页面导航
- 支持PDF缩放

### Word文档

- 支持Word文档预览
- 支持文档格式保持

### Markdown文档

- 支持Markdown渲染
- 支持代码高亮
- 支持数学公式

## 配置选项

### 显示配置

- **显示方式**: 选择显示方式（预览、下载链接等）
- **预览大小**: 设置预览区域大小
- **工具栏**: 显示/隐藏工具栏
