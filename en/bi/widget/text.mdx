# 文本

## Overview

文本微件用于在故事仪表盘中添加文本内容，支持富文本编辑，可以添加标题、段落、列表等。

## Features

- **富文本编辑**: 支持富文本编辑
- **格式设置**: 支持字体、颜色、大小等格式设置
- **链接支持**: 支持添加超链接
- **Markdown支持**: 支持Markdown语法

## Usage

### 添加文本

1. 在故事仪表盘中，点击"添加微件"
2. 选择"文本"
3. 输入文本内容
4. 设置文本格式
5. 保存配置

## 文本格式

### 基础格式

- **粗体**: 加粗文本
- **斜体**: 斜体文本
- **下划线**: 添加下划线
- **删除线**: 添加删除线

### 标题格式

- H1-H6标题
- 标题样式自定义

### 列表格式

- 有序列表
- 无序列表
- 嵌套列表
