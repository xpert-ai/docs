# 指标表格

## Overview

指标表格微件用于以表格形式展示多个指标，支持指标对比和分析。

## Features

- **多指标展示**: 支持展示多个指标
- **表格格式**: 清晰的表格格式展示
- **排序功能**: 支持按指标值排序
- **筛选功能**: 支持指标筛选

## Usage

### 添加指标表格

1. 在故事仪表盘中，点击"添加微件"
2. 选择"指标表格"
3. 选择要展示的指标
4. 配置表格列
5. 设置排序和筛选
6. 自定义样式
7. 保存配置

## 配置选项

### 指标配置

- **指标选择**: 选择要展示的指标（可多选）
- **指标顺序**: 设置指标显示顺序
- **指标名称**: 自定义指标显示名称

### 表格配置

- **列配置**: 配置表格列（指标名称、当前值、对比值等）
- **排序**: 启用列排序功能
- **筛选**: 启用筛选功能

### 样式配置

- **表格样式**: 选择表格样式
- **行样式**: 设置行样式（斑马纹等）
- **字体样式**: 设置字体大小和颜色
