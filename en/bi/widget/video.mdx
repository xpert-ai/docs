# 视频

## Overview

视频微件用于在故事仪表盘中嵌入和播放视频，支持多种视频格式和来源。

## Features

- **多格式支持**: 支持MP4、WebM、OGV等格式
- **视频源**: 支持本地视频和在线视频
- **播放控制**: 支持播放、暂停、音量等控制
- **自动播放**: 支持自动播放（可选）

## Usage

### 添加视频

1. 在故事仪表盘中，点击"添加微件"
2. 选择"视频"
3. 上传或选择视频
4. 配置视频选项
5. 设置播放选项
6. 保存配置

## 配置选项

### 视频设置

- **视频源**: 上传视频或使用URL
- **视频尺寸**: 设置视频播放器尺寸
- **封面图片**: 设置视频封面图片
- **视频描述**: 添加视频描述

### 播放选项

- **自动播放**: 启用自动播放
- **循环播放**: 启用循环播放
- **静音播放**: 默认静音播放
- **控制栏**: 显示/隐藏播放控制栏

## 支持的视频源

- 本地视频文件
- 在线视频URL
- YouTube视频（通过iframe）
- 其他视频平台
