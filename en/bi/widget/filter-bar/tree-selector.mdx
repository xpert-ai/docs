# 树型选择器

## Overview

树型选择器用于展示层级结构的数据，支持多级选择，适合展示具有父子关系的数据。

## Features

- **层级展示**: 以树形结构展示层级数据
- **多级选择**: 支持多级节点的选择
- **展开折叠**: 支持节点的展开和折叠
- **搜索功能**: 支持在树中搜索节点
- **复选框**: 支持复选框选择模式

## Usage

### 添加树型选择器

1. 在故事仪表盘中，点击"添加微件"
2. 选择"过滤器栏" -> "树型选择器"
3. 配置数据源
4. 设置层级字段
5. 配置选择模式
6. 设置显示选项
7. 保存配置

## 配置选项

### 数据配置

- **数据源**: 选择数据源连接
- **层级字段**: 设置层级字段（父级、子级）
- **显示字段**: 选择要显示的字段
- **值字段**: 选择值字段

### 选择模式

- **单选模式**: 只能选择一个节点
- **多选模式**: 可以选择多个节点
- **级联选择**: 选择父节点时自动选择子节点

### 显示选项

- **默认展开**: 设置默认展开的层级
- **显示图标**: 显示/隐藏节点图标
- **显示复选框**: 显示/隐藏复选框
- **搜索功能**: 启用/禁用搜索功能

## 使用示例

### 基本配置

```json
{
  "type": "tree-selector",
  "data": {
    "source": "department_data",
    "hierarchy": {
      "parent": "parent_id",
      "child": "id",
      "label": "name"
    }
  },
  "selection": {
    "mode": "multiple",
    "cascade": true
  },
  "display": {
    "defaultExpand": 2,
    "showCheckbox": true,
    "searchable": true
  }
}
```

## 最佳实践

1. **层级深度**: 控制层级深度，避免过深的树结构
2. **数据量**: 对于大量数据，使用懒加载
3. **选择模式**: 根据业务需求选择合适的选择模式
4. **搜索功能**: 对于大型树结构，启用搜索功能
