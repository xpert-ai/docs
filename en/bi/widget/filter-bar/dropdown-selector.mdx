# 下拉选择器

## Overview

下拉选择器提供紧凑的下拉列表选择方式，节省空间，适合在空间有限的场景中使用。

## Features

- **下拉列表**: 点击后展开下拉列表
- **搜索功能**: 支持在下拉列表中搜索
- **单选/多选**: 支持单选和多选模式
- **占位符**: 支持自定义占位符文本
- **禁用选项**: 支持禁用特定选项

## Usage

### 添加下拉选择器

1. 在故事仪表盘中，点击"添加微件"
2. 选择"过滤器栏" -> "下拉选择器"
3. 配置数据源
4. 设置选项列表
5. 配置选择模式
6. 设置显示选项
7. 保存配置

## 配置选项

### 数据配置

- **数据源**: 选择数据源或静态选项列表
- **选项字段**: 设置选项的显示文本和值
- **选项过滤**: 设置选项过滤条件

### 选择模式

- **单选**: 只能选择一个选项
- **多选**: 可以选择多个选项（显示为标签）
- **多选显示**: 设置多选时的显示方式

### 显示选项

- **占位符**: 设置占位符文本
- **搜索功能**: 启用/禁用搜索
- **清除按钮**: 显示/隐藏清除按钮
- **宽度**: 设置下拉选择器的宽度

### 样式配置

- **主题样式**: 选择主题样式
- **尺寸**: 设置选择器尺寸（小、中、大）
- **边框样式**: 设置边框样式

## 使用示例

### 基本配置

```json
{
  "type": "dropdown-selector",
  "data": {
    "source": "static",
    "options": [
      {"label": "选项1", "value": "option1"},
      {"label": "选项2", "value": "option2"},
      {"label": "选项3", "value": "option3"}
    ]
  },
  "selection": {
    "mode": "single"
  },
  "display": {
    "placeholder": "请选择",
    "searchable": true,
    "clearable": true
  }
}
```

## 最佳实践

1. **选项数量**: 对于选项较多的情况，启用搜索功能
2. **占位符**: 使用清晰的占位符文本提示用户
3. **多选显示**: 多选时合理设置显示方式，避免占用过多空间
4. **性能优化**: 对于大量选项，使用虚拟滚动
