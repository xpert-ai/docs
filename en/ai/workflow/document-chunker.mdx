# 文档分块器

## Overview

文档分块器节点用于将文档分割成多个块，便于处理和检索。

## Features

- **文档分割**: 将文档分割成块
- **分块策略**: 配置分块策略
- **块大小控制**: 控制块的大小
- **重叠处理**: 处理块之间的重叠

## Usage

1. 添加文档分块器节点
2. 选择分块策略
3. 配置分块参数
4. 执行分块
5. 处理分块结果
