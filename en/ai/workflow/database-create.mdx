# 数据库新增

## Overview

数据库新增节点用于向数据库插入新记录，支持单条和批量插入。

## Features

- **数据插入**: 插入新记录
- **批量插入**: 支持批量插入
- **数据验证**: 验证插入数据
- **错误处理**: 处理插入错误

## Usage

1. 添加数据库新增节点
2. 选择数据表
3. 配置插入数据
4. 设置验证规则
5. 执行插入操作
