# 文档源

## Overview

文档源节点用于从各种来源获取文档，支持文件上传、URL获取、数据库查询等方式。

## Features

- **多源支持**: 支持多种文档来源
- **文件上传**: 上传本地文件
- **URL获取**: 从URL获取文档
- **数据库查询**: 从数据库获取文档

## Usage

1. 添加文档源节点
2. 选择文档来源类型
3. 配置来源参数
4. 获取文档
5. 处理文档数据
