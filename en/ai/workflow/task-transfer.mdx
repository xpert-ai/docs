# 任务移交

## Overview

任务移交节点用于将任务移交给其他智能体或系统处理，实现任务分发和协作。

## Features

- **任务移交**: 移交任务给其他智能体
- **移交条件**: 配置移交条件
- **任务跟踪**: 跟踪移交任务状态
- **结果接收**: 接收移交任务结果

## Usage

1. 添加任务移交节点
2. 选择目标智能体
3. 配置移交内容
4. 设置移交条件
5. 执行任务移交
