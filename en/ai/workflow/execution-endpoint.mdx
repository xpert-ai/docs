# 执行终点

## Overview

执行终点节点用于标记工作流的结束点，定义工作流的最终输出结果。

## Features

- **流程终止**: 终止工作流执行
- **结果返回**: 返回最终结果
- **状态设置**: 设置流程状态
- **清理资源**: 清理临时资源

## Usage

1. 添加执行终点节点
2. 配置返回结果
3. 设置流程状态
4. 配置清理操作
