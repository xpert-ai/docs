# 智能体

## Overview

智能体工作流用于定义智能体的执行流程，通过工作流可以实现复杂的任务处理逻辑。

## Features

- **流程定义**: 定义智能体的执行流程
- **节点配置**: 配置工作流节点
- **条件分支**: 支持条件分支逻辑
- **循环执行**: 支持循环执行

## 工作流组成

### 节点类型

- **开始节点**: 工作流的起始节点
- **处理节点**: 执行具体处理逻辑
- **判断节点**: 条件判断节点
- **结束节点**: 工作流的结束节点

### 连接关系

- **顺序连接**: 顺序执行
- **条件连接**: 条件分支
- **并行连接**: 并行执行

## 创建工作流

### 步骤

1. 进入工作流页面
2. 点击"新建工作流"按钮
3. 添加节点
4. 配置节点
5. 连接节点
6. 测试工作流
7. 保存工作流

## 节点配置

### 处理节点

- **节点名称**: 设置节点名称
- **处理逻辑**: 配置处理逻辑
- **输入输出**: 配置输入输出

### 判断节点

- **判断条件**: 设置判断条件
- **分支配置**: 配置分支逻辑
