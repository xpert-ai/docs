# 知识检索

## Overview

知识检索节点用于从知识库中检索相关知识，为智能体提供知识支持。

## Features

- **知识检索**: 检索知识库内容
- **语义搜索**: 支持语义搜索
- **结果排序**: 对检索结果排序
- **相关性过滤**: 过滤相关性低的结果

## Usage

1. 添加知识检索节点
2. 选择知识库
3. 配置检索查询
4. 设置检索参数
5. 处理检索结果
