# 变量赋值

## Overview

变量赋值节点用于给变量赋值，支持多种赋值方式和表达式。

## Features

- **变量赋值**: 给变量赋值
- **表达式支持**: 支持表达式计算
- **类型转换**: 自动类型转换
- **批量赋值**: 支持批量赋值

## Usage

1. 添加变量赋值节点
2. 选择目标变量
3. 设置赋值表达式
4. 配置类型转换
5. 执行赋值
