# 数据库删除

## Overview

数据库删除节点用于删除数据库中的记录，支持条件删除和批量删除。

## Features

- **记录删除**: 删除数据库记录
- **条件删除**: 根据条件删除
- **批量删除**: 支持批量删除
- **安全控制**: 防止误删除

## Usage

1. 添加数据库删除节点
2. 选择数据表
3. 配置删除条件
4. 设置安全验证
5. 执行删除操作
