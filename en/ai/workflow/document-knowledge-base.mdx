# 文档知识库

## Overview

文档知识库节点用于将文档存储到知识库中，支持文档索引和检索。

## Features

- **文档存储**: 存储文档到知识库
- **索引构建**: 构建文档索引
- **向量化**: 将文档向量化
- **检索支持**: 支持文档检索

## Usage

1. 添加文档知识库节点
2. 选择知识库
3. 配置存储参数
4. 执行文档存储
5. 构建索引
