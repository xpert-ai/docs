# 变量聚合

## Overview

变量聚合节点用于聚合多个变量的值，支持多种聚合方式。

## Features

- **变量聚合**: 聚合多个变量
- **聚合方式**: 支持多种聚合方式
- **结果计算**: 计算聚合结果
- **类型处理**: 处理不同类型数据

## Usage

1. 添加变量聚合节点
2. 选择要聚合的变量
3. 选择聚合方式
4. 配置聚合参数
5. 设置输出变量
