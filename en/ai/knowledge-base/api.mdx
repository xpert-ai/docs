# API

## Overview

知识库API提供了完整的接口，允许开发者通过编程方式管理知识库。

## API端点

### 知识管理

- **POST /api/knowledge**: 创建知识
- **GET /api/knowledge/{id}**: 获取知识详情
- **PUT /api/knowledge/{id}**: 更新知识
- **DELETE /api/knowledge/{id}**: 删除知识

### 知识检索

- **POST /api/knowledge/search**: 搜索知识
- **GET /api/knowledge/list**: 获取知识列表

## 请求示例

### 创建知识

```bash
POST /api/knowledge
Content-Type: application/json

{
  "title": "知识标题",
  "content": "知识内容",
  "type": "document"
}
```
