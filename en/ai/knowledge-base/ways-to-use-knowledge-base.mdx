# 使用知识库的方式

## Overview

本文档介绍在智能体中使用知识库的多种方式，帮助您更好地利用知识库提升智能体的能力。

## 使用方式

### 1. 直接检索

智能体可以直接检索知识库获取相关信息：

```
用户: 什么是机器学习？
智能体: [检索知识库] 机器学习是...
```

### 2. 上下文增强

将知识库内容作为上下文提供给智能体：

```
上下文: [知识库相关内容]
用户问题: ...
智能体回答: ...
```

### 3. 知识增强生成

使用知识库内容增强生成的内容：

- 引用知识库内容
- 基于知识库内容生成回答
- 结合知识库和生成内容

## 配置方法

### 在智能体中启用知识库

1. 进入智能体配置
2. 选择知识库
3. 配置检索参数
4. 设置使用方式
5. 保存配置

## 最佳实践

1. **选择合适的知识**: 选择与智能体任务相关的知识
2. **优化检索**: 优化检索参数提高准确性
3. **定期更新**: 定期更新知识库内容
4. **质量监控**: 监控知识库使用效果
