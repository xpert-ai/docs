# 步骤四: 上传文件

## Overview

上传文件步骤用于将文档文件上传到知识流水线进行处理，支持多种文件格式。

## 支持的文件格式

### 文档格式

- **PDF**: PDF文档
- **Word**: Word文档（.doc, .docx）
- **Markdown**: Markdown文档
- **文本**: 纯文本文件（.txt）

### 其他格式

- **Excel**: Excel表格（.xls, .xlsx）
- **PowerPoint**: PPT文档（.ppt, .pptx）
- **HTML**: HTML文件

## 上传方法

### 1. 单文件上传

1. 点击"上传文件"按钮
2. 选择要上传的文件
3. 等待上传完成
4. 文件自动进入处理流程

### 2. 批量上传

1. 点击"批量上传"按钮
2. 选择多个文件
3. 确认上传
4. 批量处理文件

### 3. 文件夹上传

1. 选择文件夹
2. 上传整个文件夹
3. 自动处理文件夹中的所有文件

## 文件处理

### 处理状态

- **上传中**: 文件正在上传
- **处理中**: 文件正在处理
- **已完成**: 文件处理完成
- **失败**: 文件处理失败

### 查看处理结果

- **处理进度**: 查看文件处理进度
- **处理日志**: 查看处理日志
- **结果预览**: 预览处理结果
