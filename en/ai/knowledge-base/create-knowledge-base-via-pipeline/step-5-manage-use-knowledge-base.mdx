# 步骤五: 管理和使用知识库

## Overview

知识库创建完成后，需要进行管理和使用，包括知识库的维护、查询、更新等操作。

## 知识库管理

### 查看知识库

- **知识库列表**: 查看所有知识库
- **知识库详情**: 查看知识库详细信息
- **知识统计**: 查看知识库统计信息

### 知识库维护

- **更新知识**: 更新知识库中的知识
- **删除知识**: 删除不需要的知识
- **知识分类**: 对知识进行分类管理

### 知识库设置

- **基本信息**: 修改知识库基本信息
- **权限设置**: 设置知识库Access Permission
- **索引设置**: 配置知识库索引

## 使用知识库

### 查询知识

- **关键词搜索**: 通过关键词搜索知识
- **语义搜索**: 使用语义搜索查找相关知识
- **高级搜索**: 使用高级搜索功能

### 在智能体中使用

1. 配置智能体
2. 选择知识库
3. 设置检索参数
4. 启用知识库

### API使用

通过API接口使用知识库：

```bash
POST /api/knowledge/search
Content-Type: application/json

{
  "query": "搜索内容",
  "knowledge_base_id": "kb-123"
}
```

## 最佳实践

1. **定期更新**: 定期更新知识库内容
2. **质量监控**: 监控知识库质量
3. **性能优化**: 优化知识库性能
4. **Permission Management**: 合理设置访问权限
