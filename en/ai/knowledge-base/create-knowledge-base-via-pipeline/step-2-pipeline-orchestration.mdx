# 步骤二: 知识流水线编排

## Overview

知识流水线编排用于配置流水线的处理步骤和流程，定义数据在各个步骤之间的流转方式。

## 编排步骤

### 1. 添加处理步骤

流水线可以包含以下处理步骤：

- **数据采集**: 从数据源采集数据
- **数据预处理**: 清洗和预处理数据
- **文本提取**: 从文档中提取文本
- **知识提取**: 提取结构化知识
- **向量化**: 将知识转换为向量
- **索引构建**: 构建知识索引

### 2. 配置步骤参数

为每个步骤配置参数：

- **输入配置**: 配置步骤的输入
- **处理参数**: 设置处理参数
- **输出配置**: 配置步骤的输出

### 3. 设置步骤连接

- **连接步骤**: 连接各个处理步骤
- **数据流转**: 配置数据在步骤间的流转
- **条件分支**: 设置条件分支逻辑

## 编排示例

```
数据采集 → 数据预处理 → 文本提取 → 知识提取 → 向量化 → 索引构建
```
