# 维护文档

## Overview

维护文档功能用于管理和维护知识库中的文档，包括文档的更新、删除、版本管理等。

## Features

- **文档更新**: 更新文档内容
- **版本管理**: 管理文档版本
- **文档删除**: 删除不需要的文档
- **批量操作**: 支持批量操作文档

## 文档操作

### 更新文档

1. 选择要更新的文档
2. 点击"编辑"按钮
3. 修改文档内容
4. 保存更改

### 版本管理

- **Version History**: 查看文档Version History
- **Version Comparison**: 对比不同版本
- **Version Rollback**: 回滚到之前的版本
