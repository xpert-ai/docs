# 指标管理

## Overview

指标管理工具用于管理和维护业务指标，支持指标的注册、认证、分类和使用。

## Features

- **指标注册**: 注册业务指标
- **指标认证**: 认证指标质量
- **指标分类**: 对指标进行分类管理
- **指标使用**: 在分析中使用指标

## Usage

### 注册指标

1. 进入指标管理工具
2. 点击"注册指标"
3. 填写指标信息
   - 指标名称
   - 指标代码
   - 计算公式
   - 数据源
4. 提交注册

### 指标管理

- **指标列表**: 查看所有指标
- **指标详情**: 查看指标详细信息
- **指标编辑**: 编辑指标配置
- **指标删除**: 删除不需要的指标
