# 飞书

## Overview

飞书工具提供了与飞书平台的集成功能，可以在工作流中使用飞书的相关功能。

## Features

- **消息发送**: 发送飞书消息
- **群组管理**: 管理飞书群组
- **文件操作**: 操作飞书文件
- **审批流程**: 集成飞书审批流程

## Usage

### 配置飞书连接

1. 进入飞书工具配置
2. 配置飞书应用凭证
3. 授权Access Permission
4. 测试连接
5. 保存配置
