# 对话机器人

## Overview

对话机器人功能用于创建和管理智能对话机器人，提供自然语言交互能力。

## Features

- **机器人创建**: 创建对话机器人
- **对话能力**: 配置机器人的对话能力
- **知识库集成**: 集成知识库
- **多轮对话**: 支持多轮对话

## 创建机器人

### 步骤

1. 进入对话机器人页面
2. 点击"新建机器人"
3. 填写机器人信息
   - 机器人名称
   - 机器人描述
   - 机器人头像
4. 配置对话能力
5. 关联知识库（可选）
6. 保存机器人

## 机器人配置

### 对话设置

- **欢迎语**: 设置机器人欢迎语
- **回复风格**: 配置回复风格
- **对话上下文**: 设置上下文长度

### 知识库集成

- **选择知识库**: 选择机器人使用的知识库
- **检索策略**: 配置知识检索策略
- **引用显示**: 配置是否显示知识来源
