# Bash 命令执行工具

## Overview

Bash命令执行工具允许在虚拟环境中执行Bash命令，实现系统级操作和脚本执行。

## Features

- **命令执行**: 执行Bash命令
- **脚本运行**: 运行Bash脚本
- **输出捕获**: 捕获命令输出
- **错误处理**: 处理命令错误

## Usage

1. 进入虚拟环境页面
2. 选择Bash命令执行工具
3. 输入要执行的命令
4. 执行命令
5. 查看执行结果
