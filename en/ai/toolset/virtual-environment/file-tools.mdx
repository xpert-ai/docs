# 文件工具

## Overview

文件工具提供了文件Operations，包括文件读写、目录管理、文件搜索等。

## Features

- **文件读写**: 读取和写入文件
- **目录管理**: 创建和管理目录
- **文件搜索**: 搜索文件
- **文件操作**: 复制、移动、删除文件

## Usage

1. 进入虚拟环境页面
2. 选择文件工具
3. 选择文件操作类型
4. 配置操作参数
5. 执行操作
