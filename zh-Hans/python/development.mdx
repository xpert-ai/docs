---
title: '开发'
description: '在本地预览变更以更新文档'
---

<Info>
  **前置条件**：
  - Node.js 版本 19 或更高
  - 包含 `docs.json` 的文档仓库
</Info>

按照以下步骤在你的操作系统上安装并运行 Mintlify。

<Steps>
<Step title="安装 Mintlify CLI">

```bash
npm i -g mint
```
</Step>

<Step title="本地预览">

进入包含 `docs.json` 的文档目录，然后运行：

```bash
mint dev
```

文档的本地预览将在 `http://localhost:3000` 提供。

</Step>
</Steps>

## 自定义端口

默认情况下，Mintlify 使用 3000 端口。你可以通过 `--port` 参数自定义端口。例如，要使用 3333 端口：

```bash
mint dev --port 3333
```
