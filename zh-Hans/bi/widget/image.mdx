# 图片

## 概述

图片微件用于在故事仪表盘中展示图片，支持多种图片格式，可以设置图片大小、对齐方式等。

## 功能特性

- **多格式支持**: 支持JPG、PNG、GIF、SVG等格式
- **尺寸控制**: 支持自定义图片尺寸
- **对齐方式**: 支持多种对齐方式
- **链接支持**: 支持图片点击跳转

## 使用方法

### 添加图片

1. 在故事仪表盘中，点击"添加微件"
2. 选择"图片"
3. 上传或选择图片
4. 设置图片属性
5. 配置链接（可选）
6. 保存配置

## 配置选项

### 图片设置

- **图片源**: 上传图片或使用URL
- **图片尺寸**: 设置图片宽度和高度
- **对齐方式**: 设置图片对齐方式（左、中、右）
- **图片描述**: 添加图片描述（用于无障碍访问）

### 链接配置

- **链接地址**: 设置点击图片跳转的链接
- **打开方式**: 选择在新窗口或当前窗口打开
