# 平面表格

## 概述

平面表格是最基础的表格类型，以行列形式展示数据，适合展示结构化数据。

## 功能特性

- **数据展示**: 以行列形式清晰展示数据
- **列排序**: 支持按列排序
- **列筛选**: 支持列级别的筛选
- **分页显示**: 支持分页显示大量数据
- **列宽调整**: 支持调整列宽度

## 使用方法

### 添加平面表格

1. 在故事仪表盘中，点击"添加微件"
2. 选择"分析表格" -> "平面表格"
3. 配置数据源
4. 选择要显示的列
5. 设置表格样式
6. 配置交互功能
7. 保存配置

## 配置选项

### 数据配置

- **数据源**: 选择数据源连接
- **查询**: 配置数据查询语句
- **列选择**: 选择要显示的列
- **列顺序**: 设置列的显示顺序

### 表格配置

- **表头**: 显示/隐藏表头
- **行号**: 显示/隐藏行号
- **斑马纹**: 启用/禁用斑马纹样式
- **边框**: 设置表格边框样式

### 列配置

- **列宽**: 设置每列的宽度
- **列对齐**: 设置列的对齐方式（左、中、右）
- **列格式**: 设置数据格式（数字、日期、文本等）
- **列排序**: 启用/禁用列排序功能

### 交互功能

- **排序**: 启用列排序功能
- **筛选**: 启用列筛选功能
- **分页**: 启用分页功能
- **导出**: 启用导出功能（Excel、CSV等）

## 样式设置

### 表格样式

- **表格主题**: 选择表格主题样式
- **字体设置**: 设置字体类型和大小
- **颜色配置**: 设置表格颜色

### 行样式

- **行高**: 设置行高度
- **悬停效果**: 设置行悬停效果
- **选中样式**: 设置选中行的样式

## 使用示例

### 基本配置

```json
{
  "type": "flat-table",
  "data": {
    "source": "sales_data",
    "columns": ["date", "product", "amount", "quantity"]
  },
  "style": {
    "theme": "default",
    "striped": true,
    "bordered": true
  },
  "features": {
    "sortable": true,
    "filterable": true,
    "pagination": true
  }
}
```

## 最佳实践

1. **列数控制**: 避免显示过多列，影响可读性
2. **数据量**: 对于大量数据，使用分页功能
3. **列宽**: 合理设置列宽，确保内容完整显示
4. **排序筛选**: 为重要列启用排序和筛选功能
