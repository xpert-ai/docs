# iframe

## 概述

iframe微件用于在故事仪表盘中嵌入外部网页或应用，实现内容的集成展示。

## 功能特性

- **网页嵌入**: 支持嵌入外部网页
- **应用集成**: 支持嵌入其他应用
- **尺寸控制**: 支持自定义iframe尺寸
- **安全控制**: 支持安全策略配置

## 使用方法

### 添加iframe

1. 在故事仪表盘中，点击"添加微件"
2. 选择"iframe"
3. 输入要嵌入的URL
4. 设置iframe尺寸
5. 配置安全选项
6. 保存配置

## 配置选项

### URL配置

- **URL地址**: 输入要嵌入的网页URL
- **URL验证**: 验证URL的有效性

### 尺寸配置

- **宽度**: 设置iframe宽度
- **高度**: 设置iframe高度
- **响应式**: 启用响应式尺寸

### 安全配置

- **沙箱模式**: 启用沙箱模式
- **允许脚本**: 允许/禁止脚本执行
- **允许表单**: 允许/禁止表单提交

## 注意事项

- 确保嵌入的网页支持iframe嵌入
- 注意跨域安全策略
- 建议使用HTTPS协议
