# 日历选择器

## 概述

日历选择器用于选择日期或日期范围，提供直观的日历界面，适合日期相关的数据筛选。

## 功能特性

- **日期选择**: 选择单个日期
- **日期范围**: 选择日期范围
- **月份视图**: 以月份视图显示日历
- **快捷选项**: 提供快捷日期选项（今天、昨天、本周、本月等）
- **日期格式**: 支持自定义日期格式

## 使用方法

### 添加日历选择器

1. 在故事仪表盘中，点击"添加微件"
2. 选择"过滤器栏" -> "日历选择器"
3. 选择选择模式（单日期/日期范围）
4. 配置日期格式
5. 设置快捷选项
6. 配置显示选项
7. 保存配置

## 配置选项

### 选择模式

- **单日期**: 选择单个日期
- **日期范围**: 选择开始日期和结束日期
- **多日期**: 选择多个不连续的日期

### 日期格式

- **显示格式**: 设置日期的显示格式（YYYY-MM-DD、MM/DD/YYYY等）
- **值格式**: 设置日期值的格式
- **语言**: 设置日历语言

### 快捷选项

- **预设范围**: 提供预设的日期范围选项
  - 今天
  - 昨天
  - 本周
  - 上周
  - 本月
  - 上月
  - 本季度
  - 本年
- **自定义快捷选项**: 添加自定义的快捷选项

### 限制选项

- **最小日期**: 设置可选择的最小日期
- **最大日期**: 设置可选择的最大日期
- **禁用日期**: 设置禁用的日期或日期规则

### 显示选项

- **显示方式**: 内联显示或弹出显示
- **显示时间**: 是否显示时间选择
- **周起始日**: 设置一周的起始日（周一或周日）

## 使用示例

### 基本配置

```json
{
  "type": "calendar-selector",
  "mode": "range",
  "format": {
    "display": "YYYY-MM-DD",
    "value": "YYYY-MM-DD"
  },
  "presets": [
    "today",
    "yesterday",
    "thisWeek",
    "thisMonth"
  ],
  "restrictions": {
    "minDate": "2020-01-01",
    "maxDate": "2024-12-31"
  },
  "display": {
    "style": "popup",
    "showTime": false,
    "weekStart": "monday"
  }
}
```

## 最佳实践

1. **日期格式**: 使用统一的日期格式，符合用户习惯
2. **快捷选项**: 提供常用的快捷选项，提高用户体验
3. **日期限制**: 根据业务需求设置合理的日期限制
4. **时区处理**: 注意时区问题，确保日期选择的准确性
