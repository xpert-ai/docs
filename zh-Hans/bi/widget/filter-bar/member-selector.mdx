# 成员选择器

## 概述

成员选择器用于从维度成员中选择特定的成员值，支持单选和多选，适合维度数据的筛选。

## 功能特性

- **成员列表**: 显示维度成员列表
- **单选/多选**: 支持单选和多选模式
- **搜索功能**: 支持搜索成员
- **分组显示**: 支持按分组显示成员
- **快速选择**: 支持全选、反选等快速操作

## 使用方法

### 添加成员选择器

1. 在故事仪表盘中，点击"添加微件"
2. 选择"过滤器栏" -> "成员选择器"
3. 选择维度
4. 配置选择模式
5. 设置显示选项
6. 配置默认值
7. 保存配置

## 配置选项

### 维度配置

- **维度选择**: 选择要使用的维度
- **成员来源**: 配置成员数据来源
- **成员过滤**: 设置成员过滤条件

### 选择模式

- **单选**: 只能选择一个成员
- **多选**: 可以选择多个成员
- **多选限制**: 设置最多选择的成员数量

### 显示选项

- **显示方式**: 列表、下拉、标签等
- **搜索功能**: 启用/禁用搜索
- **分组显示**: 启用/禁用分组显示
- **排序方式**: 设置成员排序方式

### 默认值

- **默认选择**: 设置默认选中的成员
- **默认全部**: 默认选择所有成员
- **默认空**: 默认不选择任何成员

## 使用示例

### 基本配置

```json
{
  "type": "member-selector",
  "dimension": "product",
  "selection": {
    "mode": "multiple",
    "maxSelection": 10
  },
  "display": {
    "style": "list",
    "searchable": true,
    "grouped": true
  },
  "default": {
    "values": ["product1", "product2"]
  }
}
```

## 最佳实践

1. **成员数量**: 对于成员数量较多的情况，启用搜索功能
2. **选择限制**: 设置合理的选择数量限制
3. **默认值**: 根据业务需求设置合适的默认值
4. **性能优化**: 对于大量成员，使用虚拟滚动
