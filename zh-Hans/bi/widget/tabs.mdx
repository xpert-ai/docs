# 选项卡

## 概述

选项卡微件用于创建选项卡式的内容组织方式，可以在同一区域内切换显示不同的内容。

## 功能特性

- **多标签页**: 支持多个标签页
- **内容切换**: 支持点击标签切换内容
- **样式定制**: 支持自定义选项卡样式
- **响应式布局**: 支持响应式布局

## 使用方法

### 添加选项卡

1. 在故事仪表盘中，点击"添加微件"
2. 选择"选项卡"
3. 添加标签页
4. 为每个标签页添加内容
5. 配置选项卡样式
6. 保存配置

## 配置选项

### 标签页配置

- **标签名称**: 设置标签页名称
- **标签图标**: 添加标签图标（可选）
- **标签内容**: 添加标签页内容

### 样式配置

- **选项卡样式**: 选择选项卡样式（顶部、底部、左侧、右侧）
- **标签样式**: 自定义标签样式
- **激活样式**: 设置激活标签的样式
