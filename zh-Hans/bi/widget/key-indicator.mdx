# 关键指标

## 概述

关键指标微件用于展示重要的业务指标，以卡片形式突出显示关键数据，帮助用户快速了解业务状况。

## 功能特性

- **指标展示**: 清晰展示关键业务指标
- **趋势显示**: 显示指标变化趋势
- **对比功能**: 支持与上期、去年同期对比
- **样式定制**: 支持自定义卡片样式

## 使用方法

### 添加关键指标

1. 在故事仪表盘中，点击"添加微件"
2. 选择"关键指标"
3. 选择要展示的指标
4. 配置指标计算方式
5. 设置对比基准
6. 自定义样式
7. 保存配置

## 配置选项

### 指标配置

- **指标选择**: 选择要展示的指标
- **计算方式**: 设置指标的计算方式
- **时间范围**: 设置指标的时间范围

### 对比配置

- **对比类型**: 选择对比类型（环比、同比等）
- **对比基准**: 设置对比基准期
- **显示方式**: 设置对比结果的显示方式

### 样式配置

- **卡片样式**: 选择卡片样式
- **颜色配置**: 设置指标颜色
- **图标配置**: 设置指标图标
