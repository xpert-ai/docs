# 参数控制器

## 概述

参数控制器用于接收用户输入并创建动态参数，支持多种输入类型，可以与其他微件联动，实现动态数据查询和分析。

## 功能特性

- **多种输入类型**: 支持文本、数字、日期、选择等输入类型
- **参数绑定**: 将输入值绑定到参数
- **数据验证**: 支持输入数据验证
- **联动功能**: 支持与其他微件联动
- **默认值**: 支持设置默认值

## 使用方法

### 添加参数控制器

1. 在故事仪表盘中，点击"添加微件"
2. 选择"输入控制器" -> "参数控制器"
3. 选择输入类型
4. 配置输入选项
5. 设置验证规则
6. 绑定参数
7. 配置联动关系
8. 保存配置

## 输入类型

### 文本输入

- **单行文本**: 单行文本输入框
- **多行文本**: 多行文本输入框
- **密码输入**: 密码输入框（隐藏输入）

### 数字输入

- **整数**: 整数输入
- **小数**: 小数输入
- **范围输入**: 数字范围输入（最小值、最大值）

### 日期输入

- **日期选择器**: 日期选择器
- **日期范围选择器**: 日期范围选择器
- **时间选择器**: 时间选择器

### 选择输入

- **下拉选择**: 下拉选择框
- **单选按钮**: 单选按钮组
- **复选框**: 复选框组

## 配置选项

### 参数绑定

- **参数名称**: 设置参数名称
- **参数类型**: 设置参数类型（文本、数字、日期等）
- **参数值**: 绑定输入值到参数

### 验证规则

- **必填验证**: 设置是否必填
- **格式验证**: 设置格式验证规则
- **范围验证**: 设置数值范围验证
- **自定义验证**: 设置自定义验证规则

### 联动配置

- **联动微件**: 选择要联动的微件
- **联动方式**: 设置联动方式（筛选、刷新、参数传递等）
- **联动条件**: 设置联动触发条件

### 显示选项

- **标签**: 设置输入框标签
- **占位符**: 设置占位符文本
- **提示信息**: 设置提示信息
- **帮助文本**: 设置帮助文本

## 使用示例

### 基本配置

```json
{
  "type": "parameter-controller",
  "inputType": "number",
  "label": "销售额阈值",
  "parameter": {
    "name": "sales_threshold",
    "type": "number"
  },
  "validation": {
    "required": true,
    "min": 0,
    "max": 1000000
  },
  "default": 10000,
  "linkage": {
    "targets": ["sales_chart"],
    "action": "filter",
    "condition": "value > 0"
  }
}
```

## 参数使用

### 在查询中使用

在数据查询中使用参数：

```sql
SELECT * FROM sales 
WHERE amount > @sales_threshold
AND date >= @start_date
```

### 在计算中使用

在计算字段中使用参数：

```
Profit Margin = (Revenue - Cost) / Revenue * @margin_multiplier
```
