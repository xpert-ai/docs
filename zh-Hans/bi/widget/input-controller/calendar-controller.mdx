# 日历控制器

## 概述

日历控制器用于接收用户的日期输入，支持单个日期和日期范围选择，可以与其他微件联动，实现基于日期的数据查询和分析。

## 功能特性

- **日期选择**: 支持选择单个日期
- **日期范围**: 支持选择日期范围
- **快捷选项**: 提供快捷日期选项（今天、昨天、本周、本月等）
- **数据验证**: 支持日期格式验证和范围限制
- **联动功能**: 支持与其他微件联动

## 使用方法

### 添加日历控制器

1. 在故事仪表盘中，点击"添加微件"
2. 选择"输入控制器" -> "日历控制器"
3. 选择输入类型（单日期/日期范围）
4. 配置日期选项
5. 设置验证规则
6. 配置联动关系
7. 保存配置

## 输入类型

### 单日期选择

- **日期选择器**: 选择单个日期
- **日期格式**: 设置日期显示格式
- **默认值**: 设置默认日期

### 日期范围选择

- **开始日期**: 选择开始日期
- **结束日期**: 选择结束日期
- **范围限制**: 设置日期范围限制

## 配置选项

### 日期格式

- **显示格式**: 设置日期的显示格式（YYYY-MM-DD、MM/DD/YYYY等）
- **值格式**: 设置日期值的格式
- **语言**: 设置日历语言

### 快捷选项

- **预设范围**: 提供预设的日期范围选项
  - 今天
  - 昨天
  - 本周
  - 上周
  - 本月
  - 上月
  - 本季度
  - 本年
- **自定义快捷选项**: 添加自定义的快捷选项

### 验证规则

- **必填验证**: 设置是否必填
- **日期范围**: 设置可选择的日期范围
- **禁用日期**: 设置禁用的日期或日期规则

### 联动配置

- **联动微件**: 选择要联动的微件
- **联动方式**: 设置联动方式（筛选、刷新等）

## 使用示例

### 基本配置

```json
{
  "type": "calendar-controller",
  "mode": "range",
  "format": {
    "display": "YYYY-MM-DD",
    "value": "YYYY-MM-DD"
  },
  "presets": [
    "today",
    "yesterday",
    "thisWeek",
    "thisMonth"
  ],
  "validation": {
    "required": true,
    "minDate": "2020-01-01",
    "maxDate": "2024-12-31"
  },
  "linkage": {
    "targets": ["chart1", "chart2"],
    "action": "filter"
  }
}
```
