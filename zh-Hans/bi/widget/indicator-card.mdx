# 指标卡片

## 概述

指标卡片微件用于以卡片形式展示单个指标，突出显示指标值和变化趋势。

## 功能特性

- **指标展示**: 清晰展示指标值和单位
- **趋势显示**: 显示指标变化趋势（上升/下降）
- **对比显示**: 显示与基准期的对比
- **样式定制**: 支持自定义卡片样式

## 使用方法

### 添加指标卡片

1. 在故事仪表盘中，点击"添加微件"
2. 选择"指标卡片"
3. 选择要展示的指标
4. 配置指标显示方式
5. 设置趋势和对比
6. 自定义样式
7. 保存配置

## 配置选项

### 指标配置

- **指标选择**: 选择要展示的指标
- **指标名称**: 显示指标名称
- **指标值**: 显示指标当前值
- **单位**: 显示指标单位

### 趋势配置

- **趋势显示**: 启用趋势显示
- **趋势类型**: 选择趋势类型（箭头、图标等）
- **趋势颜色**: 设置上升/下降的颜色

### 样式配置

- **卡片样式**: 选择卡片样式
- **背景颜色**: 设置卡片背景颜色
- **字体样式**: 设置字体大小和颜色
