# 指标注册

## 概述

指标注册功能允许您将业务指标注册到数据分析平台，统一管理和使用。通过指标注册，可以确保指标定义的标准化和一致性，提高数据分析的准确性和效率。

## 注册指标

### 注册步骤

1. 进入指标管理页面
2. 点击"注册指标"按钮
3. 填写指标基本信息
   - **指标名称**: 业务指标的中文名称，要求清晰明确
   - **指标代码**: 唯一的指标标识符，建议使用英文和数字
   - **指标描述**: 指标的详细说明，包括业务含义、计算逻辑等
   - **计算公式**: 指标的计算公式或SQL表达式
   - **数据源**: 指标依赖的数据源
4. 设置指标属性
   - **指标类型**: 选择指标类型（聚合指标、计算指标等）
   - **单位**: 指标的单位（元、个、百分比等）
   - **精度**: 数值的精度设置
   - **业务域**: 选择指标所属的业务域
5. 设置指标权限
   - 选择指标的可见范围
   - 设置编辑权限
6. 提交注册申请

### 指标注册示例

```json
{
  "name": "月销售额",
  "code": "MONTHLY_SALES",
  "description": "月度销售总额",
  "formula": "SUM(sales.amount)",
  "dataSource": "sales_db",
  "type": "aggregate",
  "unit": "元",
  "precision": 2,
  "category": "销售域"
}
```

## 指标信息

### 基本信息

- **指标名称**: 业务指标的中文名称，用于展示和识别
- **指标代码**: 唯一的指标标识符，用于系统内部引用
- **指标描述**: 指标的详细说明，帮助用户理解指标含义
- **计算公式**: 指标的计算逻辑，支持SQL和自定义公式
- **数据源**: 指标依赖的数据源连接

### 计算公式

支持多种计算方式：

#### SQL表达式
```sql
SELECT SUM(amount) FROM sales WHERE month = '2024-01'
```

#### 聚合函数
- **SUM**: 求和
- **AVG**: 平均值
- **COUNT**: 计数
- **MAX**: 最大值
- **MIN**: 最小值

#### 自定义公式
支持使用数学运算符和函数构建复杂公式：
```
(销售额 - 成本) / 销售额 * 100
```

## 指标审核

注册的指标需要经过审核才能正式使用：

### 审核流程

1. **提交注册申请**: 填写完整的指标信息后提交
2. **管理员审核**: 管理员检查指标定义的准确性和合理性
3. **审核反馈**: 如有问题，管理员会反馈修改意见
4. **修改完善**: 根据反馈修改指标定义
5. **审核通过**: 审核通过后指标正式生效，可以在系统中使用

### 审核要点

- 指标定义的准确性
- 计算公式的正确性
- 数据源的可用性
- 命名规范的符合性
- 业务域的合理性

## 指标类型

### 聚合指标

基于数据聚合的指标，如：
- 总销售额
- 平均订单金额
- 客户总数

### 计算指标

基于其他指标计算得出的指标，如：
- 销售增长率 = (本期销售额 - 上期销售额) / 上期销售额
- 利润率 = (利润 / 收入) × 100%
