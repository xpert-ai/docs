## 概述

功能模块管理用于配置租户可用的功能模块，控制租户的功能访问权限。

## 功能模块列表

### 数据分析模块

- 语义模型
- 指标管理
- 故事仪表盘

### 微件模块

- 分析卡片
- 分析表格
- 过滤器栏
- 输入控制器

### 网站功能模块

- 数据源管理
- 单点登录
- 企业租户管理

## 配置方法

1. 进入功能模块页面
2. 选择租户
3. 启用/禁用功能模块
4. 设置模块权限
5. 保存配置
