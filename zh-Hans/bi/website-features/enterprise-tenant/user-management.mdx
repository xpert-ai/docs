# 用户管理

## 概述

用户管理用于管理租户内的用户，包括用户的创建、编辑、删除和权限分配。

## 用户操作

### 添加用户

1. 点击"添加用户"按钮
2. 填写用户信息
   - 用户名
   - 邮箱
   - 显示名称
   - 密码
3. 分配角色
4. 分配组织
5. 保存用户

### 编辑用户

1. 选择要编辑的用户
2. 修改用户信息
3. 更新角色和组织
4. 保存更改

### 删除用户

1. 选择要删除的用户
2. 点击"删除"按钮
3. 确认删除操作

## 用户信息

- **用户名**: 用户的登录名
- **邮箱**: 用户邮箱地址
- **显示名称**: 用户的显示名称
- **角色**: 用户所属的角色
- **组织**: 用户所属的组织
