# 服务器代理

## 概述

服务器代理用于配置服务器级别的代理设置，用于访问外部资源或解决网络访问问题。

## 功能特性

- **HTTP代理**: 配置HTTP代理服务器
- **HTTPS代理**: 配置HTTPS代理服务器
- **SOCKS代理**: 配置SOCKS代理服务器
- **代理池**: 支持多个代理服务器

## 配置步骤

### 添加服务器代理

1. 进入代理页面
2. 点击"服务器代理"菜单
3. 点击"新建代理"按钮
4. 选择代理类型
5. 填写代理信息
   - 代理服务器地址
   - 端口号
   - 用户名（如果需要）
   - 密码（如果需要）
6. 测试代理连接
7. 保存配置

## 配置选项

### 代理类型

- **HTTP代理**: 用于HTTP请求的代理服务器
- **HTTPS代理**: 用于HTTPS请求的代理服务器
- **SOCKS代理**: 支持SOCKS4和SOCKS5协议

### 代理设置

- **服务器地址**: 代理服务器的IP地址或域名
- **端口号**: 代理服务器的端口号
- **认证方式**: 选择认证方式（无/基本认证）
- **用户名**: 代理认证用户名
- **密码**: 代理认证密码

## 使用场景

- 访问受限的外部资源
- 解决网络访问问题
- 提高访问速度
- 实现负载均衡
