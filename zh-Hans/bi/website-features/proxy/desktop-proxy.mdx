# 桌面代理

## 概述

桌面代理用于配置桌面应用程序的代理设置，用于桌面客户端访问外部资源。

## 功能特性

- **桌面应用代理**: 为桌面应用配置代理
- **系统代理**: 配置系统级代理设置
- **应用代理**: 为特定应用配置代理
- **代理切换**: 支持代理切换和管理

## 配置步骤

### 添加桌面代理

1. 进入代理页面
2. 点击"桌面代理"菜单
3. 点击"新建代理"按钮
4. 选择代理类型
5. 填写代理信息
6. 配置应用绑定
7. 测试代理连接
8. 保存配置

## 配置选项

### 代理类型

- **HTTP代理**: HTTP代理配置
- **HTTPS代理**: HTTPS代理配置
- **SOCKS代理**: SOCKS代理配置

### 应用绑定

- **系统代理**: 应用到系统所有应用
- **应用代理**: 应用到特定应用
- **规则代理**: 基于规则的代理配置

## 使用场景

- 桌面应用访问外部API
- 桌面客户端数据同步
- 跨网络访问资源
- 安全访问控制
