# 自建应用配置

## 概述

自建应用配置用于配置自定义的ChatBI应用，包括应用信息、功能设置、权限配置等。

## 功能特性

- **应用创建**: 创建自定义应用
- **功能配置**: 配置应用功能
- **权限管理**: 管理应用权限
- **应用发布**: 发布应用供使用

## 配置步骤

### 1. 创建应用

1. 进入自建应用配置页面
2. 点击"新建应用"
3. 填写应用基本信息
   - 应用名称
   - 应用描述
   - 应用图标
4. 保存应用

### 2. 配置功能

- **数据源配置**: 配置应用使用的数据源
- **对话配置**: 配置对话能力
- **界面配置**: 配置应用界面
- **集成配置**: 配置第三方集成
