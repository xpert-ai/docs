# 幻灯片工具

## 概述

幻灯片工具用于创建和管理演示文稿，支持幻灯片创建、编辑和导出。

## 功能特性

- **幻灯片创建**: 创建演示文稿
- **内容编辑**: 编辑幻灯片内容
- **模板使用**: 使用幻灯片模板
- **导出功能**: 导出幻灯片文件

## 使用方法

1. 进入虚拟环境页面
2. 选择幻灯片工具
3. 创建新幻灯片
4. 编辑内容
5. 保存或导出
