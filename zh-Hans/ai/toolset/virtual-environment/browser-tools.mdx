# Browser 工具

## 概述

Browser工具提供了基础的浏览器功能，包括网页访问、内容提取等。

## 功能特性

- **网页访问**: 访问网页
- **内容提取**: 提取网页内容
- **截图功能**: 网页截图
- **PDF生成**: 生成PDF文件

## 使用方法

1. 进入虚拟环境页面
2. 选择Browser工具
3. 输入网页URL
4. 执行操作
5. 查看结果
