# Python 执行工具

## 概述

Python执行工具允许在虚拟环境中执行Python代码，支持代码运行、调试和结果查看。

## 功能特性

- **代码执行**: 执行Python代码
- **交互式执行**: 支持交互式Python环境
- **包管理**: 管理Python包
- **结果展示**: 展示执行结果

## 使用方法

1. 进入虚拟环境页面
2. 选择Python执行工具
3. 输入Python代码
4. 执行代码
5. 查看执行结果
