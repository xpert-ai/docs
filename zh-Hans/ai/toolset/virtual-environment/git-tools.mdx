# GIT 工具

## 概述

GIT工具提供了Git版本控制功能，包括代码提交、分支管理、合并等。

## 功能特性

- **代码提交**: 提交代码变更
- **分支管理**: 管理Git分支
- **代码合并**: 合并代码分支
- **版本历史**: 查看版本历史

## 使用方法

1. 进入虚拟环境页面
2. 选择GIT工具
3. 选择Git操作
4. 配置参数
5. 执行操作
