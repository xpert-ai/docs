# Crawl4AI

## 概述

Crawl4AI是一个强大的网页爬取工具，支持从网页中提取结构化数据，适用于数据采集和信息提取场景。

## 功能特性

- **网页爬取**: 爬取网页内容
- **数据提取**: 提取结构化数据
- **JavaScript渲染**: 支持JavaScript渲染的网页
- **数据清洗**: 自动清洗提取的数据

## 使用方法

### 配置Crawl4AI

1. 进入MCP工具页面
2. 选择Crawl4AI工具
3. 配置工具参数
4. 测试工具连接
5. 保存配置

### 使用示例

```python
# 爬取网页内容
result = crawl4ai.crawl(url="https://example.com")
print(result.content)
```
