# 飞书文档转换策略

## 概述

飞书文档转换策略用于定义如何将飞书文档转换为系统可用的格式，包括格式转换、内容提取等。

## 功能特性

- **格式转换**: 转换文档格式
- **内容提取**: 提取文档内容
- **结构化处理**: 处理文档结构
- **质量保证**: 保证转换质量

## 配置步骤

1. 进入飞书文档转换策略配置
2. 选择转换格式
3. 配置转换规则
4. 设置内容提取规则
5. 配置质量检查
6. 保存策略
