# 飞书文档源策略

## 概述

飞书文档源策略用于定义如何从飞书系统中获取文档，包括文档选择、过滤、排序等策略。

## 功能特性

- **文档选择**: 选择要获取的文档
- **过滤策略**: 配置文档过滤规则
- **排序策略**: 配置文档排序方式
- **批量获取**: 支持批量获取文档

## 配置步骤

1. 进入飞书文档源策略配置
2. 选择文档来源
3. 配置过滤条件
4. 设置排序规则
5. 配置获取参数
6. 保存策略
