# 问题分类器

## 概述

问题分类器节点用于对用户问题进行分类，根据问题类型路由到不同的处理流程。

## 功能特性

- **问题分类**: 对问题进行自动分类
- **分类规则**: 配置分类规则
- **路由控制**: 根据分类结果路由
- **多分类支持**: 支持多标签分类

## 使用方法

1. 添加问题分类器节点
2. 配置分类规则
3. 设置分类类别
4. 配置路由逻辑
5. 连接后续节点
