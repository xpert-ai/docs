# 回答

## 概述

回答节点用于生成和返回用户问题的答案，支持多种回答格式。

## 功能特性

- **答案生成**: 生成答案内容
- **格式支持**: 支持多种格式
- **模板使用**: 使用答案模板
- **动态内容**: 支持动态内容

## 使用方法

1. 添加回答节点
2. 配置答案内容
3. 选择答案格式
4. 设置模板变量
5. 连接输出
