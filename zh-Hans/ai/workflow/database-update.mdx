# 数据库更新

## 概述

数据库更新节点用于更新数据库中的记录，支持条件更新和批量更新。

## 功能特性

- **记录更新**: 更新数据库记录
- **条件更新**: 根据条件更新
- **批量更新**: 支持批量更新
- **更新验证**: 验证更新结果

## 使用方法

1. 添加数据库更新节点
2. 选择数据表
3. 配置更新条件
4. 设置更新数据
5. 执行更新操作
