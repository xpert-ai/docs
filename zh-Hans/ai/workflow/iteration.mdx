# 迭代

## 概述

迭代节点用于对集合数据进行迭代处理，支持循环执行和批量操作。

## 功能特性

- **循环执行**: 循环执行操作
- **集合处理**: 处理集合数据
- **索引访问**: 访问迭代索引
- **中断控制**: 支持中断循环

## 使用方法

1. 添加迭代节点
2. 选择要迭代的数据源
3. 配置迭代逻辑
4. 设置循环条件
5. 连接循环体节点
