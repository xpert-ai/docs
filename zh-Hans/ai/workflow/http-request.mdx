# HTTP 请求

## 概述

HTTP请求节点用于发送HTTP请求，调用外部API或服务。

## 功能特性

- **HTTP请求**: 发送HTTP请求
- **多种方法**: 支持GET、POST等
- **请求头配置**: 配置请求头
- **响应处理**: 处理响应结果

## 使用方法

1. 添加HTTP请求节点
2. 配置请求URL
3. 选择请求方法
4. 设置请求参数
5. 配置请求头
6. 处理响应结果
