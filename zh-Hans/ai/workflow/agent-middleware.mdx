# 智能体中间件

## 概述

智能体中间件节点用于在工作流中应用中间件，实现请求拦截、日志记录、错误处理等功能。

## 功能特性

- **请求拦截**: 拦截和处理请求
- **日志记录**: 记录执行日志
- **错误处理**: 统一错误处理
- **性能监控**: 监控执行性能

## 使用方法

1. 添加智能体中间件节点
2. 选择中间件类型
3. 配置中间件参数
4. 设置执行顺序
5. 应用中间件
