# 智能体工具

## 概述

智能体工具节点用于调用智能体的工具能力，实现智能体与工具的集成。

## 功能特性

- **工具集成**: 集成智能体工具
- **自动调用**: 智能体自动调用工具
- **工具选择**: 智能体选择合适工具
- **结果处理**: 处理工具执行结果

## 使用方法

1. 添加智能体工具节点
2. 选择智能体
3. 配置工具权限
4. 设置调用策略
5. 执行工具调用
