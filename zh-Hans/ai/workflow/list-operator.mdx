# 列表操作器

## 概述

列表操作器节点用于对列表数据进行各种操作，包括过滤、映射、排序等。

## 功能特性

- **列表过滤**: 过滤列表元素
- **列表映射**: 映射列表元素
- **列表排序**: 排序列表元素
- **列表聚合**: 聚合列表数据

## 使用方法

1. 添加列表操作器节点
2. 选择操作类型
3. 配置操作参数
4. 设置输入输出
5. 执行操作
