# 注释

## 概述

注释节点用于在工作流中添加说明性注释，提高工作流的可读性和可维护性。

## 功能特性

- **添加注释**: 添加文本注释
- **格式化**: 支持格式化文本
- **链接支持**: 支持添加链接
- **文档说明**: 提供文档说明

## 使用方法

1. 添加注释节点
2. 输入注释内容
3. 格式化注释文本
4. 保存注释
