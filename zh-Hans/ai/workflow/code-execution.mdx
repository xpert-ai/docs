# 代码执行

## 概述

代码执行节点用于在工作流中执行自定义代码，实现复杂的业务逻辑。

## 功能特性

- **代码执行**: 执行自定义代码
- **多语言支持**: 支持多种编程语言
- **变量访问**: 访问工作流变量
- **结果返回**: 返回执行结果

## 使用方法

1. 添加代码执行节点
2. 选择编程语言
3. 编写代码
4. 配置输入输出变量
5. 执行代码
