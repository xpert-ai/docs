# 工具集

## 概述

工具集节点用于在工作流中调用工具集中的工具，实现功能扩展。

## 功能特性

- **工具调用**: 调用工具集中的工具
- **工具选择**: 选择要使用的工具
- **参数配置**: 配置工具参数
- **结果处理**: 处理工具返回结果

## 使用方法

1. 添加工具集节点
2. 选择工具集
3. 选择具体工具
4. 配置工具参数
5. 执行工具调用
