# 文档图像理解

## 概述

文档图像理解节点用于理解和提取文档图像中的内容，支持OCR和图像分析。

## 功能特性

- **OCR识别**: 识别图像中的文字
- **图像分析**: 分析图像内容
- **表格识别**: 识别表格结构
- **版面分析**: 分析文档版面

## 使用方法

1. 添加文档图像理解节点
2. 选择图像来源
3. 配置识别参数
4. 执行图像理解
5. 提取内容
