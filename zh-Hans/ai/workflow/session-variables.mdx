# 会话变量

## 概述

会话变量节点用于管理和使用会话级别的变量，在整个会话过程中保持数据状态。

## 功能特性

- **变量定义**: 定义会话变量
- **变量读取**: 读取会话变量值
- **变量更新**: 更新会话变量
- **作用域管理**: 管理变量作用域

## 使用方法

1. 添加会话变量节点
2. 定义变量名称和类型
3. 设置初始值
4. 在工作流中使用变量
