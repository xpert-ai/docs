# 数据库自定义SQL

## 概述

数据库自定义SQL节点用于执行自定义SQL语句，实现灵活的数据库操作。

## 功能特性

- **SQL执行**: 执行自定义SQL
- **参数绑定**: 支持参数绑定
- **结果处理**: 处理SQL执行结果
- **事务支持**: 支持事务操作

## 使用方法

1. 添加数据库自定义SQL节点
2. 选择数据库连接
3. 编写SQL语句
4. 配置参数
5. 执行SQL
