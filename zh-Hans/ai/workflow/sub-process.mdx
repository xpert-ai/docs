# 子流程

## 概述

子流程节点用于调用其他工作流作为子流程，实现工作流的模块化和复用。

## 功能特性

- **流程调用**: 调用其他工作流
- **参数传递**: 传递参数给子流程
- **结果接收**: 接收子流程返回结果
- **流程复用**: 实现流程复用

## 使用方法

1. 添加子流程节点
2. 选择要调用的工作流
3. 配置输入参数
4. 设置输出变量
5. 执行子流程
