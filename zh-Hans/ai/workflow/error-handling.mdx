# 错误处理

## 概述

错误处理节点用于捕获和处理工作流执行过程中的错误，确保工作流的健壮性。

## 功能特性

- **错误捕获**: 捕获执行错误
- **错误分类**: 分类不同类型的错误
- **错误处理**: 执行错误处理逻辑
- **错误日志**: 记录错误日志

## 使用方法

1. 添加错误处理节点
2. 配置错误捕获规则
3. 设置错误处理逻辑
4. 配置错误通知
5. 设置错误恢复策略
