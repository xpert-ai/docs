# 工具调用

## 概述

工具调用节点用于直接调用工具，支持多种工具类型和调用方式。

## 功能特性

- **工具调用**: 调用各种工具
- **动态调用**: 支持动态工具选择
- **批量调用**: 支持批量调用工具
- **结果聚合**: 聚合多个工具结果

## 使用方法

1. 添加工具调用节点
2. 选择工具类型
3. 配置调用参数
4. 设置结果处理
5. 执行工具调用
