# 命令

## 概述

命令功能用于配置AI供应商的命令设置，包括命令定义、命令执行、命令权限等。

## 功能特性

- **命令定义**: 定义AI供应商的命令
- **命令执行**: 执行配置的命令
- **命令权限**: 设置命令执行权限
- **命令日志**: 记录命令执行日志

## 配置步骤

### 1. 定义命令

1. 进入AI智能助理页面
2. 选择"配置AI供应商"
3. 进入"命令"配置
4. 点击"新建命令"
5. 填写命令信息
   - 命令名称
   - 命令描述
   - 命令内容
   - 执行参数
6. 保存命令

### 2. 配置命令权限

- **执行权限**: 设置谁可以执行命令
- **参数验证**: 配置参数验证规则
- **执行超时**: 设置命令执行超时时间

## 使用方法

### 执行命令

1. 在工作流中选择命令节点
2. 选择要执行的命令
3. 配置命令参数
4. 执行命令
5. 查看执行结果
