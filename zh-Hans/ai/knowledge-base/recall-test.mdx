# 召回测试

## 概述

召回测试用于测试知识库的检索效果，评估知识检索的准确性和相关性。

## 功能特性

- **测试查询**: 执行测试查询
- **结果评估**: 评估检索结果
- **性能分析**: 分析检索性能
- **优化建议**: 提供优化建议

## 测试方法

### 创建测试用例

1. 进入召回测试页面
2. 点击"新建测试用例"
3. 输入测试查询
4. 设置期望结果
5. 保存测试用例

### 执行测试

1. 选择测试用例
2. 点击"执行测试"
3. 查看测试结果
4. 分析结果质量

## 评估指标

### 准确率

- **精确率**: 检索结果中相关结果的比例
- **召回率**: 检索到的相关结果占所有相关结果的比例
- **F1分数**: 精确率和召回率的调和平均
