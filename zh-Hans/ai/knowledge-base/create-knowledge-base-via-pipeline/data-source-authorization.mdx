# 数据源授权

## 概述

数据源授权用于配置知识流水线访问数据源的权限，确保数据安全访问。

## 授权类型

### API授权

- **API密钥**: 使用API密钥授权
- **OAuth授权**: 使用OAuth协议授权
- **Token授权**: 使用Token授权

### 数据库授权

- **用户名密码**: 使用用户名密码授权
- **证书授权**: 使用证书授权
- **连接字符串**: 使用连接字符串授权

## 配置步骤

### 1. 选择数据源

1. 进入数据源授权页面
2. 选择要授权的数据源
3. 选择授权类型

### 2. 配置授权信息

- **授权凭证**: 填写授权凭证
- **权限范围**: 设置权限范围
- **有效期**: 设置授权有效期

### 3. 测试授权

- **测试连接**: 测试授权连接
- **验证权限**: 验证访问权限
- **保存授权**: 保存授权配置

## 授权管理

### 查看授权

- **授权列表**: 查看所有授权
- **授权详情**: 查看授权详细信息
- **授权状态**: 查看授权状态

### 更新授权

- **更新凭证**: 更新授权凭证
- **更新权限**: 更新权限范围
- **续期授权**: 续期授权有效期

### 撤销授权

- **撤销授权**: 撤销不需要的授权
- **批量撤销**: 批量撤销授权

## 安全建议

1. **定期更新**: 定期更新授权凭证
2. **最小权限**: 使用最小权限原则
3. **安全存储**: 安全存储授权信息
4. **访问监控**: 监控授权访问情况
