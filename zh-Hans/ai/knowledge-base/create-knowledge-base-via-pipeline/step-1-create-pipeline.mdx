# 步骤一: 创建知识流水线

## 概述

创建知识流水线是使用知识流水线创建知识库的第一步，需要配置流水线的基本信息和处理流程。

## 创建步骤

### 1. 进入知识流水线页面

1. 进入知识库页面
2. 点击"通过知识流水线创建知识库"
3. 点击"创建知识流水线"按钮

### 2. 填写基本信息

- **流水线名称**: 设置流水线的名称
- **流水线描述**: 描述流水线的用途
- **目标知识库**: 选择或创建目标知识库

### 3. 配置数据源

- **数据源类型**: 选择数据源类型（文件、API、数据库等）
- **数据源配置**: 配置数据源连接信息

### 4. 设置处理流程

- **处理步骤**: 选择处理步骤
- **步骤顺序**: 设置步骤的执行顺序
- **步骤参数**: 配置每个步骤的参数
