Claude Code 是 Anthropic 官方的 CLI 工具。本指南将帮助你设置 Claude Code 以协助编写和维护文档。

## 前置条件

- 有效的 Claude 订阅（Pro、Max 或 API 访问）

## 安装步骤

1. 全局安装 Claude Code：

  ```bash
  npm install -g @anthropic-ai/claude-code
```

2. 进入你的文档目录。
3. （可选）将下方的 `CLAUDE.md` 文件添加到项目中。
4. 运行 `claude` 启动。

## 创建 `CLAUDE.md`

在你的文档仓库根目录创建 `CLAUDE.md`，用于训练 Claude Code 遵循你的文档规范：

````markdown
# Mintlify 文档

## 协作方式
- 可以对想法提出质疑，这可能会带来更好的文档。请引用来源并说明理由
- 遇到不清楚的地方一定要先提问，不要做假设
- 绝不要撒谎、猜测或杜撰信息

## 项目背景
- 格式：带 YAML frontmatter 的 MDX 文件
- 配置：docs.json 负责导航、主题与设置
- 组件：Mintlify 组件

## 内容策略
- 内容要恰到好处，既不冗长也不缺失
- 优先保证信息准确性与可用性
- 尽可能让内容长期有效
- 添加新内容前先搜索已有内容。除非有策略理由，否则避免重复
- 检查已有模式保持一致性
- 从最小可行的变更开始

## 页面 Frontmatter 要求
- title：清晰、描述性的页面标题
- description：用于 SEO/导航的简洁摘要

## 写作标准
- 第二人称（“你”）
- 操作步骤前先列出前置条件
- 发布前测试所有代码示例
- 与现有页面风格与格式保持一致
- 包含基础与高级用法
- 所有代码块标注语言标签
- 所有图片添加 alt 文本
- 站内链接使用相对路径

## Git 工作流
- 提交时绝不使用 --no-verify
- 开始之前询问如何处理未提交的更改
- 没有明确分支时创建新分支
- 开发过程中频繁提交
- 不要跳过或禁用 pre-commit hooks

## 不要做
- 在 MDX 文件中省略 frontmatter
- 站内链接使用绝对 URL
- 包含未经测试的代码示例
- 做假设——必须先澄清
````
